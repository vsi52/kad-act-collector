> Расширение помогает последовательно получить документы из открытого пользователем дела и локально подготовить их для работы. Оно не является официальным сервисом судебной системы и не даёт права на массовое копирование базы, обход ограничений или неправомерное распространение документов.

Если пользователь не согласен с условиями, он должен отказаться от
продолжения и удалить расширение. Использование новой редакции

## 2. Назначение и принцип работы

Расширение предназначено для работы с конкретным делом, которое
пользователь самостоятельно нашёл и открыл на `kad.arbitr.ru`. По
команде пользователя оно получает метаданные доступных документов,
последовательно открывает выбранные PDF для штатного скачивания
браузером и может локально извлечь текст из уже скачанных PDF для
создания единого TXT.

Документы, текст, номера дел и история работы не направляются на сервер
разработчика. Расширение не предоставляет юридические консультации, не
оценивает достоверность документов и не заменяет проверку оригинала.

Расширение является независимым программным продуктом. Оно не

`kad.arbitr.ru` и Google Chrome используются только для описания
совместимости и назначения продукта; права на соответствующие
обозначения принадлежат их правообладателям.

Пользователь обязуется:

- получать только те документы, к которым он имеет законный доступ;
- работать в пределах открытого им дела и разумного объёма, необходимого
  для конкретной задачи;
- соблюдать законодательство, права изготовителя базы данных, авторские

- самостоятельно обеспечить законность хранения, анализа, передачи и

  требуется законом, и учитывать ограничения на персональные данные,

Официальная сборка не предназначена для массового формирования
собственной копии Картотеки арбитражных дел, систематического сбора
тысяч дел, фонового мониторинга всей базы, продажи массива данных или
предоставления производного API. Пользователь не вправе использовать
официальную сборку для обхода CAPTCHA, блокировок IP, авторизации, кодов

создания чрезмерной нагрузки.

При HTTP 403/429, HTML-странице защиты или ином признаке ограничения

обычного доступа. Случайные интервалы и пакетные паузы предназначены для
снижения нагрузки, а не для сокрытия автоматизации или обхода защиты.

поддержку допускается только по инициативе и с явного согласия

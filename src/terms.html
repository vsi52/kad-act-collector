2. Назначение и принцип работы

Расширение предназначено для работы с конкретным делом, которое пользователь самостоятельно нашёл и открыл на kad.arbitr.ru. По команде пользователя оно получает метаданные доступных документов, последовательно открывает выбранные PDF для штатного скачивания браузером и может локально извлечь текст из уже скачанных PDF для создания единого TXT.

Документы, текст, номера дел и история работы не направляются на сервер разработчика. Расширение не предоставляет юридические консультации, не оценивает достоверность документов и не заменяет проверку оригинала.

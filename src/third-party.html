Сторонние компоненты

Расширение «Сборщик судебных актов для kad.arbitr.ru»

Установочная сборка использует PDF.js для локального извлечения текстового слоя из выбранных пользователем PDF.

PDF.js

  • Пакет: pdfjs-dist
  • Версия: 5.4.149
  • Проект: github.com/mozilla/pdf.js
  • Лицензия: Apache License 2.0
  • Назначение: локальное извлечение текстового слоя из выбранных пользователем PDF.

Полный текст лицензии включён в установочную сборку как PDFJS_LICENSE.txt.

Инструменты сборки

Исходный проект собирается с помощью esbuild версии 0.25.8, распространяемого по лицензии MIT. esbuild не включается в исполняемую установочную сборку расширения.

Собственный код проекта распространяется по условиям файла LICENSE.

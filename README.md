# Сборщик судебных актов для kad.arbitr.ru

Независимое расширение Chrome Manifest V3 для работы с документами одного дела, открытого пользователем на `kad.arbitr.ru`.

Расширение получает доступный список документов, позволяет последовательно сохранить выбранные PDF либо локально извлечь текстовый слой и сформировать единый TXT в заданной хронологии.

> Расширение не является официальным продуктом `kad.arbitr.ru`, АО «ПравоТех», арбитражных судов или иных государственных органов и не связано с ними.

![Рабочее окно расширения](screenshots/02_Act_Collector_1280x800.png)

## Возможности

- сбор документов только из открытого пользователем дела;
- выбор отдельных документов, определений или всего доступного списка;
- последовательное скачивание PDF с понятными именами;
- создание единого TXT с датами, типами документов и ссылками на оригиналы;
- локальная обработка PDF посредством PDF.js;
- остановка очереди при ограничении доступа со стороны сайта;
- отсутствие собственного сервера, аналитики и рекламы.

Расширение не предназначено для массового копирования базы КАД, не обходит CAPTCHA, блокировки и другие ограничения доступа. Доступ к документу не означает права на его публичное распространение.

## Установка

### Готовая версия

Скачайте [`KAD_Act_Collector_v0.1.20.zip`](release/KAD_Act_Collector_v0.1.20.zip), распакуйте архив, откройте `chrome://extensions`, включите режим разработчика и выберите «Загрузить распакованное расширение».

Архив также подготовлен для загрузки в Chrome Web Store.

### Сборка из исходников

Требуются Node.js 20 или новее и npm:

```bash
npm ci
npm run build
npm run check
```

Собранное расширение появится в каталоге `dist`. Команда `npm run package` дополнительно создаст установочный и исходный ZIP в каталоге `release`.

## Как пользоваться

1. Откройте карточку конкретного дела на `kad.arbitr.ru`.
2. Нажмите «Собрать судебные акты» на странице или значок расширения.
3. При первом запуске прочитайте и примите пользовательское соглашение.
4. Выберите документы, порядок и результат: PDF или единый TXT.
5. Не закрывайте исходную вкладку до завершения операции.

Для стабильного сохранения PDF в Chrome рекомендуется выбрать скачивание PDF вместо открытия в браузере: `chrome://settings/content/pdfDocuments`.
